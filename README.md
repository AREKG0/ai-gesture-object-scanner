# AI Gesture Controller + Object Safety Scanner

Real-time AI project built with Python, MediaPipe, YOLOv3 and OpenCV.

## Features
- Control your computer with hand gestures (mouse, volume, screenshot, media)
- Real-time object detection using YOLOv3 (80 object types)
- Automatic danger alerts for sharp objects and fire detection
- Beep sound warnings for dangerous objects

## Tech Stack
Python · MediaPipe · YOLOv3 · OpenCV · PyAutoGUI · NumPy

## How to run
1. pip install -r requirements.txt
2. Download yolov3.weights from https://pjreddie.com/media/files/yolov3.weights
3. python gesture_controller.py
4. python object_scanner.py
